# 개발 편의

## webextension-reloader

- Web Extension 은 변경된 파일을 바로 보여주지 않음
  - popup 은 바로 반영
  - background, content_scripts 는 바로 반영하지 않음
- `yarn run webextension-reloader:watch`
- Chrome `chrome://extensions` 에서 Load Unpacked
- Extension ID 와 Web App Host 등록
- `Option + R` 로 새로고침
